# Interaction Design

## Goals
- Make every action understandable before and after it happens.

## Guidelines
- Show clear focus states and active states for every interactive element.
- Use labels, helper text, and validation messages that tell users what happens next.
- Design loading, empty, success, and failure states as part of the main flow.
- Keep touch targets large enough for mobile use.

## Avoid
- Invisible focus.
- Buttons that do not look clickable.
- Error messages that only say something went wrong.
